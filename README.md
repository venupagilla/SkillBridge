# SkillBridge

SkillBridge is a full-stack web application designed to bridge the gap between job seekers and employers by identifying skill gaps, verifying skills through simulations, and providing a dynamic learning roadmap.

## Tech Stack

-   **Frontend**: React, TailwindCSS
-   **Backend**: FastAPI, Python
-   **Database**: PostgreSQL
-   **AI**: Mocked / Gemini

## Getting Started

### Backend

1.  Navigate to `backend/`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Run server: `uvicorn main:app --reload`

### Frontend

1.  Navigate to `frontend/`
2.  Install dependencies: `npm install`
3.  Run server: `npm run dev`
